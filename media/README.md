# media/

Placera bilder och videofiler här som ska visas på webbplatsen.

Rekommendationer:

- Använd webbvänliga format: JPEG/WEBP för bilder, MP4/WebM för video.
- Namnge filer tydligt, t.ex. `solnedgang-2025-07-15.jpg` eller `skogstur-1.mp4`.
- Förstora inte bilder i onödan — skapa optimerade thumbnails för snabbare laddning.

Användning i projektet:

Den här versionen av sajten visar en plats för media i `index.html`.
För att automatiskt visa filer kan du lägga till ett litet scripts steg som genererar en `media/index.json` med fil-listan under bygg/deploy.
